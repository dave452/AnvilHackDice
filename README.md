# Asset naming

  -Images/Resources
		>File name:
			ScreenName-screenobject.png
			e.g Home-Background.png
			e.g DiceShow-Outline.png
		> Identifier:
			First 5 characters of LETTERSOFSCREEN_OJECT
			e.g HOME_BACKG
			e.g DICES_OUTLI
		
	
	-Script files
		> Functions:
			funFunctionName.c
			e.g funTextUpdate.c
		> Screens
			scrScreenName.c
			e.g scrHome.c
		> File Containing data such as numbers
			datNameOfDataList.c
			e.g datAppUUID.c
